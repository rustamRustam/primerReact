Пример работы на React!

Цель:

Данная работа выполнена для показа моих возможностей в разработке на React. Задачи разработать дизайн не ставилось, по этому дизайн выполнен на низком уровне. Но понятность интерфейса на должном уровне. Верстка выполнена с адаптивностью, т.е. комфорт сохраняется и на смартфоне и на ноутбуке и на персональном компьютере.

Задание:

Разработать одно-страничный сайт (муляж сайта) по формированию личной коллекции наиболее понравившихся картин из представленного каталога.

Описание от разработчика

Разработка выполнена в нескольких вариациях. Основное отличие вариантов состоит в виде использования компонентов React: функциональные компонентах и компоненты классы. Далее к каждому варианту добавляются технология Redax.
Весь код доступен на gitHub в репозитории.(URL репозитория: https://github.com/rustamRustam/primerReact)
